# gold-svg

Download monthly gold prices (DKK per gram) from Nordisk Guld and render **only** the graph line as an SVG.

## Usage

```bash
python gold_svg.py --start 2020-01-01 --end 2026-04-01 --output gold.svg
```

Optional:

```bash
python gold_svg.py -o gold.svg --width 1200 --height 600 --stroke-width 2 --pad 10
```
